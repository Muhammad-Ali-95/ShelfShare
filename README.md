 # Book Exchange App

A full-stack web application built with **Next.js** that allows users to exchange, buy, and sell books within a community.

## Tech Stack

* **Frontend:** Next.js (React Framework), Tailwind CSS
* **Backend:** Node.js, Express (Hosted separately)
* **Icons:** React Icons
* **HTTP Client:** Axios

## Features

* **User Authentication:** Signup and Login pages.
* **Dashboard:** Manage your listed books.
* **Book Exchange:** Browse books available for exchange.
* **Cart System:** Add books to cart for checkout.
* **Community:** Connect with other book lovers.
* **Responsive Design:** Styled with Tailwind CSS for mobile and desktop.
